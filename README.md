# AGENCY_OS

一个本地优先的 AI 媒体生成工作台：统一管理图像/视频生成模型、参考素材、资产、提示词反推与生成画布。

> **仍在持续完善。** 当前版本已有不少可用能力，但模型适配、接口兼容性、错误提示和工作流体验仍需要继续优化，欢迎通过 Issue 提出问题与建议。

## 功能

- 图像与视频生成：通过已配置的服务商模型提交任务。
- 参考素材：上传、粘贴或从资产库添加多张图像/视频参考，并调整顺序。
- 资产库：查看、下载、复制图片、复用提示词、将资产送入创建页或画布。
- AI 反推：本地规则回退与可选视觉模型辅助，生成可继续编辑的提示词。
- 生成画布：拖入资产，组合多个参考节点后发起图像或视频生成。
- 提示词库、队列、用量与中英文界面。

## 技术栈

- React 19 + TypeScript + Vite
- React Router
- React Flow (`@xyflow/react`)
- Anime.js

## 快速开始

### 前置条件

- Node.js 20 或更高版本
- npm 10 或更高版本

### 安装与启动

```bash
git clone https://github.com/<你的用户名>/agency-os.git
cd agency-os
npm ci
npm run dev
```

启动后访问终端显示的本地地址（默认通常为 `http://localhost:5173`）。

### 生产构建

```bash
npm run build
npm run preview
```

### 常用命令

```bash
npm run dev      # 本地开发
npm run build    # TypeScript 检查并构建
npm run lint     # 代码检查
npm run preview  # 预览构建产物
```

## 使用流程

1. 在 **服务商** 页面填写 API 基础 URL、密钥并同步/添加模型。
2. 前往 **创建**，选择图像或视频模型，输入提示词；按模型能力添加参考素材。
3. 在 **资产** 中查看结果，可下载、复制、复用或作为参考图继续创作。
4. 需要拆解参考图时，使用 **反推** 生成提示词，再送往创建或画布。
5. 需要组合多张素材时，使用 **画布** 把资产连接到图像/视频生成节点。

## 数据与安全

- 服务商配置、密钥、资产索引和工作区状态仅保存在当前浏览器的本地存储中；不会被写入本仓库。
- `public/generated/`、本地测试记录、截图和常见环境变量文件均已被 `.gitignore` 排除。
- 请勿将真实密钥写入源码、截图、Issue、提交信息或 `.env.example`。
- 本项目的开发代理仅用于本地开发。若部署到公网，请先实现服务端密钥托管、访问控制、限流与日志脱敏；不要把用户 API 密钥暴露给浏览器或静态站点。
